### 🚀 What I'm Building

#### 🎙️ [agentforce-voice-eval](https://github.com/afigueiredo/agentforce-voice-eval) *(coming soon)*
**Automated quality evaluation framework for Agentforce Voice agents**

Two-phase pipeline: conversation runner (parallel sessions via REST API + latency measurement) → LLM-as-Judge (Claude evaluates 5 quality dimensions with written reasoning). Catches regressions before they reach callers.

`Agentforce` `Voice AI` `LLM Eval` `Python` `Claude`

Watermark-based incremental sync from Salesforce Data Cloud to S3, with Athena query support. Handles governor limits, duplicate detection, and backfill. Built from real customer deployments.

`Data Cloud` `AWS S3` `Athena` `Python` `Salesforce`
